# Pilot Plan

- **Design Partners**: 1 electric utility, 1 pipeline/fiber operator (commitment obtained from biz-dev team).
- **Scope**: 200–500 parcels across 3 jurisdictions with varying condemnation rules.
- **KPIs**: ≥30% faster pre-petition settlements, 0 missed statutory deadlines, 100% binder completeness.
- **Data Share**: Partners provide sample title packets, appraisals, historical notices, sanitized address data.
- **Timeline**:
  - Month 0: sandbox provisioning, data ingestion dry run.
  - Month 1: landowner invites live, agent workbench pilot.
  - Month 2: counsel controls + binder exports, full evidence chain validated.
  - Month 3: go/no-go for general availability.
- **Success Exit Criteria**: KPI hit, no critical security incidents, counsel sign-off on rules + templates.
